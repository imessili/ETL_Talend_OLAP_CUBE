Le fichier Readme.txt indique les noms des deux membres (binôme) ainsi que les instructions à suivre pour lancer l'application Web.

Le projet comprend une pile de containers permettant de lancer une application de visualisation de données, en se concentrant sur les données de ventes étudiées lors de la première phase ETL.

La pile de containers comprend :

- Une base de données MongoDB contenant les données de ventes.

- Un serveur Apache.

- Un container GraphQL contenant le code du serveur et les résolveurs.
